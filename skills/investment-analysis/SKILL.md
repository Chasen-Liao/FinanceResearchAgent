# 投资分析技能

## 概述
提供专业的投资分析方法论，包括技术分析和基本面分析。

## 何时使用
- 用户要求分析股票/市场
- 需要生成投资建议
- 进行技术指标计算
- 评估股票投资价值

## 分析框架

### 技术分析

1. **趋势分析**
   - 移动平均线 (MA/EMA): 5日、10日、20日、60日、120日、250日
   - 布林带 (Bollinger Bands): 判断价格波动范围

2. **动量指标**
   - RSI (相对强弱指数): 超买超卖判断 (70以上超买, 30以下超卖)
   - MACD (指数平滑异同移动平均线): 趋势转折信号
   - KDJ (随机指标): 超买超卖和金叉死叉

3. **成交量分析**
   - OBV (能量潮): 成交量与价格趋势配合
   - 成交量变化率: 放量/缩量判断

### 基本面分析

1. **估值指标**
   - P/E (市盈率): 股票估值水平
   - P/B (市净率): 净资产估值
   - PEG (市盈率相对盈利增长比率): 成长性估值

2. **盈利能力**
   - ROE (净资产收益率): 股东回报
   - ROA (总资产收益率): 资产效率
   - 毛利率、净利率: 盈利能力

3. **成长能力**
   - 营收增速: 收入增长
   - 利润增速: 盈利增长
   - 现金流增长: 现金创造力

4. **财务健康**
   - 资产负债率: 财务杠杆
   - 流动比率: 短期偿债能力
   - 利息保障倍数: 债务偿还能力

## 输出格式

1. **技术分析报告**
   - 关键指标数值
   - 趋势判断（上涨/下跌/震荡）
   - 买卖信号识别
   - 支撑位/阻力位

2. **基本面分析报告**
   - 核心财务指标
   - 估值水平评估
   - 竞争优势分析
   - 风险因素

3. **综合建议**
   - 投资评级（买入/持有/卖出）
   - 目标价格区间
   - 风险提示
   - 持有期限建议
